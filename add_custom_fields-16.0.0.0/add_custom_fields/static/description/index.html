Screenshots

01

Custom Dynamic Fields Tree View

Once you have completed the installation of the "add_custom_fields" module in your Odoo 15 instance, proceed to access the tree view of custom dynamic fields by clicking on the "Add custom dynamic fields" menu option.

02

Custom Dynamic Fields Form View

To create a custom field, start by clicking the "Create" button located above the treeview. This action will open a form where you can input the necessary information.

03

Create Field Button

After inputting all the required data for custom fields, select the "create field" button to incorporate the field into your preferred form view.

04

Added Field

Once you click the "Add Field" button, your custom field will be placed in your chosen location.
